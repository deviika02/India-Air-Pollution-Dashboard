# 🌫️ India Air Pollution Monitoring Dashboard | Power BI

## 📌 Project Overview
An interactive Power BI dashboard analyzing real-time Air Quality Index (AQI) 
data across India (2023–2025), covering 203 cities, 27 states, and 411 
monitoring stations.

## 🗂️ Dataset
- **Source:** Kaggle — Real-Time Air Quality Index (AQI) India (2023–2025)
- **Pollutants Covered:** PM2.5, PM10, NO₂, SO₂, CO, NH₃, O₃

## 🛠️ Tools & Skills Used
| Tool | Purpose |
|------|---------|
| Power BI | Dashboard Design & Visualization |
| Power Query | Data Cleaning & Transformation |
| DAX | KPI Measures & Calculations |

## 🔄 Steps Followed
1. Imported dataset using Power Query
2. Corrected data types & handled missing values
3. Removed duplicate records
4. Created DAX measures for KPIs
5. Built interactive one-page dashboard

## 📈 Key Insights
- 🔴 **Baddi (HP)** is the most polluted city with avg pollution of 174
- 🌫️ **PM10 & PM2.5** are the dominant pollutants across India
- 🏙️ **Greater Noida, Ludhiana & Gwalior** are consistently high-pollution cities
- 📍 **411 stations** across 27 states highlight the scale of the air quality crisis

## 📌 Visuals Used
- KPI Cards, Bar Chart, Donut Chart, Treemap, Map, Table, Slicers
